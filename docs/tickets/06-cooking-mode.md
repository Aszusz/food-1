# 06 - Cooking Mode

**Type:** AFK
**Blocked by:** [04 - Recipe Create, View, and Edit](04-recipe-create-view-edit.md)

## What To Build

Add Cooking Mode from Recipe Detail. It presents the recipe's ingredient checklist and one large ordered cooking step at a time with previous and next controls. The checklist is a cooking-session convenience and must not modify the recipe or shopping-list purchase status. Close and Finish Cooking return to the originating Recipe Detail.

## Acceptance Criteria

- [ ] Start Cooking opens Cooking Mode for the selected recipe.
- [ ] Cooking Mode displays every recipe ingredient as an unchecked checklist item.
- [ ] A member can navigate the ordered steps using Previous and Next, with sensible first and last boundaries.
- [ ] Ingredient checks remain visible while navigating steps during the session.
- [ ] Close and Finish Cooking return to the recipe detail screen.
- [ ] Cooking does not change the recipe ingredients or shopping-list checked state.
